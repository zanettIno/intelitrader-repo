# InteliTrader - quero ser
Este repositório contém as soluções para dois desafios de programação propostos pela InteliTrader, implementados em linguagem C.

## Desafios Resolvidos

### 1. Menor Distância de Dois Arrays
Implementação do algoritmo para encontrar a menor distância entre elementos de dois arrays.

[Link do desafio](https://github.com/intelitrader/quero-ser/blob/master/exercicios/menor_distancia_array.md)

### 2. Livro de Ofertas
Solução para o problema de gerenciamento de livro de ofertas (order book).

[Link do desafio](https://github.com/intelitrader/quero-ser/blob/master/exercicios/livro_de_ofertas.md)

## Como Executar

### OnlineGDB
1. Acesse [OnlineGDB](https://www.onlinegdb.com/)
2. Selecione a linguagem **C**
3. Copie e cole o código do desafio desejado
4. Clique em **Run** para compilar e executar
